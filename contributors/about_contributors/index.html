About Contributors

Roles that contribute to Corpusim

  • Community Member : Connect with other contributors. Most other roles will also be members of the community.
  • Developer : Contribute to the software development (design, coding, management, etc.) effort.
  • Documentation Writer : Write or edit the content of the present website, Corpusim Docs
  • Outreach : Help our project find users, funder, and other contributors.

Any way you want to help would be appreciated!
